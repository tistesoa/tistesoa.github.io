My Projects

Docfis

Docfis is a project responsible to emit a Brazilian receipt for Americanas retail chains, using a comprehensive AWS approach. It incorporated microservices, saga pattern, event-driven architecture, and a single-page application for the web solution. It handled over 500,000 requests per day from more than 12,000 machine clients across Brazil. Notably, it operated seamlessly in offline mode, enhancing project resilience and reliability. Our cloud technology stack featured Java with the Spring Web Flux framework for reactive microservices, Node.js for serverless functions, SQS for messaging, and DynamoDB as the database solution. For offline functionality, we utilized Java with Quarkus and SQLite to create a lightweight embedded service. On the front end, Angular 12 with Material was employed. The entire infrastructure was orchestrated using Terraform.

Cards

In this project I joined a team responsible for all credit card demands for C6Bank, using the Kotlin language alongside the Javalin web framework to craft lightweight REST endpoints atop the Jetty server. We incorporated Koin for dependency injection, utilized the Fuel library for HTTP networking to facilitate communication between resources, and adopted Expose, a lightweight SQL library for seamless access to a PostgreSQL database. Beyond the tech stack,the team also employed the saga-pattern to address significant challenges using queues and asynchronous solutions.

PPE

PPE portal was responsible to handled CRUD operations, protocols, petitions, and all lawsuit processes related to the state of Rio Grande do Sul in South Brazil. Initilally built in JavaEE, with Weblogic Java application server and Oracle as database, it was migrated to newer tech at that time, using microservices built in Java 8 with CDI and Angular 2 with Boostrap for frontend.

Bry Applet

Developed a Applet responsibled for read Digital Certificates and performing digital signatures using Bry SDK libraries. This Applet could ran in Linux environment, loading of mandatory libraries on demand.
